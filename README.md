# Hospital Management System

A full-stack Hospital Management System built using React, Django REST Framework, and SQLite.

## Features
- User Registration & Login (JWT Authentication)
- Doctors Management
- Departments Management
- Patients Management
- Appointment Booking
- Dashboard

## Tech Stack
### Frontend
- React
- Vite
- Axios
- Tailwind CSS

### Backend
- Django
- Django REST Framework
- SQLite
- JWT Authentication

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd 23B1801
npm install
npm run dev
```
